# Sneaker Images for Badge System

## How to Add Sneaker Images

1. Download transparent PNG images of each sneaker from sources like:
   - **pngall.com/air-jordan-png/** - Air Jordan collection
   - **toppng.com** - Various Nike, Kyrie, KD, LeBron sneakers
   - **pngwing.com** - Free transparent PNGs
   - **freepngimg.com** - Basketball shoes

2. Name each file to match the sneaker ID exactly:

## Cartoonizing with Replicate / Nano Banana Pro

After downloading the real sneaker images:

1. Go to **nanabanana.pro** or use **Replicate API**
2. Upload each sneaker image
3. Use a prompt like:
   - "Convert to cartoon illustration style, vibrant colors, clean lines, transparent background"
   - "Stylized cartoon sneaker, bold colors, vector art style, no background"
4. Download the cartoonized version
5. Replace the original PNG with the cartoon version

## Image Requirements

- **Format**: PNG with transparent background
- **Size**: At least 500x500 pixels recommended
- **Orientation**: Side profile view (showing the full sneaker)
- **Style**: After cartoonizing, should have clean lines and vibrant colors
