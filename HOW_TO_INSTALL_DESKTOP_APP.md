## 🚀 Installation Steps

### Step 1: Double-Click the .dmg File

1. In the Finder window, double-click `SHOTIQ Basketball Analysis_0.1.0_aarch64.dmg`
2. A new window will open showing the app icon
3. **Drag the app icon to your Applications folder**

### Step 2: Open the App

**⚠️ IMPORTANT - First Time Opening:**

macOS will show a security warning because the app isn't signed. Here's how to open it:

**Method 1: Right-Click to Open**
1. Open your Applications folder
2. Find "SHOTIQ Basketball Analysis"
3. **Right-click** (or Control-click) the app
4. Select **"Open"**
5. Click **"Open"** again in the security dialog
6. The app will now open!

**Method 2: Use Terminal Command**
```bash
xattr -cr "/Applications/SHOTIQ Basketball Analysis.app"
open "/Applications/SHOTIQ Basketball Analysis.app"
```

After the first time, you can open it normally!

## 🔧 Troubleshooting

### "App is damaged and can't be opened"

Run this command to remove the security quarantine:
```bash
xattr -cr "/Applications/SHOTIQ Basketball Analysis.app"
```

Then try opening again.

### "App can't be opened because the developer cannot be verified"

1. **Right-click** the app in Applications
2. Select **"Open"**
3. Click **"Open"** in the dialog

This only needs to be done once!

### App won't launch?

Check if it's already running:
1. Look in the Dock for the app icon
2. Or press `Cmd + Tab` to see running apps
